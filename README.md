Este repositório contém uma série de automações desenvolvidas para simplificar as tarefas administrativas diárias. 
Utilizando principalmente a biblioteca Python PyAutoGui, os projetos automatizam processos repetitivos, 
resultando em uma otimização significativa do tempo de trabalho. 
Os scripts abrangem uma variedade de necessidades específicas que, se realizadas manualmente, consumiriam consideravelmente mais tempo. 
Embora muitos projetos inicialmente tenham sido configurados para meu ambiente de trabalho pessoal, 
estou adaptando-os para serem utilizados por outros colaboradores do setor, ajustando as configurações para as resoluções mais comuns utilizadas no departamento.
